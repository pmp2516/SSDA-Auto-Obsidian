# Reading System

> [!note] Area Definition
> Ongoing system for choosing, reading, and extracting value from books and articles. No end date.

## Current System

### Book selection
1. Queue lives in Notion. Sources: Lenny's Newsletter recs, [[People/Denise-Park]] mentions, [[People/Elena Vasquez]] suggestions, Goodreads friends.
2. Max 3 books "in progress" at once — one non-fiction, one fiction, one technical.
3. New books need a 10-page test before fully committing.

### Reading method
- Physical books: pen annotations in margins + dog-ear pages with key ideas.
- Kindle: highlights auto-sync to Readwise → exported weekly to [[Resources/Book-Notes]].
- Articles: saved to Omnivore → highlighted → synced to Obsidian.

### Processing
- After each non-fiction book: write a 500-word synthesis note in [[Resources/Book-Notes]].
- Pull 1–3 ideas worth turning into [[../../vault-2-Zettelkasten/20-Permanent/]] permanent notes.
- Tag ideas relevant to active projects.
