## Production Workflow

1. Read book → capture notes in [[Resources/Book-Notes]]
2. Write episode script using [[Templates/Podcast-Episode-Template]]
3. Record in home studio (Rode NT1 + Focusrite Scarlett)
4. Edit in Descript → export 320kbps MP3
5. Publish to Buzzsprout, cross-post to Substack
